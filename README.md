# YSTI - 原神人格类型指标

**Yuanshen Type Indicator** — 30 道题，测出你在提瓦特大陆的真实灵魂。

## 💡 简介

YSTI 是一款以原神 (Genshin Impact) 为主题的趣味人格测试，灵感来源于 [SBTI 赛博人格测试](https://github.com/UnluckyNinja/SBTI-test)。

通过 10 个维度、30 道精心设计的问题，将你的人格与 16 种原神角色类型进行匹配。

## 🎮 测试维度

| 维度 | 说明 |
|------|------|
| D1 冒险精神 | 你对未知世界的探索欲望 |
| D2 战斗意志 | 面对挑战时的坚持程度 |
| D3 社交主动性 | 主动与人交往的倾向 |
| D4 团队协作 | 在团队中的配合意愿 |
| D5 情感深度 | 对角色和剧情的情感投入 |
| D6 理想主义 | 为信念而战的意愿 |
| D7 规则遵守 | 对秩序和契约的态度 |
| D8 决断力 | 做决定的果断程度 |
| D9 执行力 | 任务完成的坚持度 |
| D10 自我认同 | 对自我价值的认知 |

## 🧬 角色类型

钟离型、胡桃型、刻晴型、温迪型、甘雨型、魈型、纳西妲型、达达利亚型、绫华型、可莉型、雷电将军型、流浪者型、七七型、芙宁娜型、没钱的钟离型、应急食品型
